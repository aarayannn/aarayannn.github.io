Panduan Penggunaan
Cara Menggunakan
  1. Sesuaikan parameter pertumbuhan pada tabel jika diperlukan
  2. Masukkan informasi WiFi dan Blynk (opsional)
  3. Pilih tipe board ESP dan sesuaikan konfigurasi pin
  4. Hubungkan ESP ke komputer melalui USB
  5. Klik "Hubungkan ke ESP" dan pilih port serial yang sesuai
  6. Pilih mode upload dan minggu/hari pertumbuhan saat ini
  7. Klik "Generate & Upload Firmware" untuk mengunggah firmware ke ESP
Persyaratan
  • Browser yang mendukung Web Serial API (Chrome, Edge, atau Opera)
  • ESP8266 atau ESP32 yang terhubung ke komputer melalui USB
  • Driver CH340/CP210x terinstal di komputer Anda
  • Sensor dan aktuator yang terhubung sesuai dengan konfigurasi pin
Tips
  • Mode "Parameter Mingguan" mengupload parameter untuk setiap minggu dan ESP akan otomatis menyesuaikan berdasarkan hari
  • Mode "Parameter Harian" mengupload parameter untuk setiap hari dari 25 hari siklus pertumbuhan
  • Mode "Hanya Hari/Minggu Saat Ini" hanya mengupload parameter untuk hari atau minggu yang sedang berjalan
  • Parameter akan disimpan di EEPROM ESP sehingga tetap tersimpan meskipun ESP restart
  • Gunakan Blynk untuk monitoring jarak jauh (opsional)
  • Pastikan sensor dikalibrasi dengan benar sebelum digunakan
