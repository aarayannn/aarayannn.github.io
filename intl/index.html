Panduan Penggunaan

Cara Menggunakan

  1. Input Manual:
    • Catat data PPM dan pH dari Blynk
    • Masukkan data (pisahkan dengan koma)
    • Klik "Hitung Parameter Optimal"
  2. Import CSV:
    • Export data dari Blynk format CSV
    • Pastikan ada kolom: ppm, ph
    • Upload file dan pilih minggu
    • Klik "Analisis Data CSV"
  3. Konfigurasi ESP:
    • Isi konfigurasi WiFi dan Blynk
    • Masukkan parameter target
    • Generate dan upload konfigurasi
  4. Gunakan hasil sebagai setpoint di Blynk/Arduino
  5. Ulangi setiap minggu

Algoritma Decision Tree

Sistem menggunakan algoritma pohon keputusan untuk:

  • Menganalisis stabilitas parameter
  • Mendeteksi anomali dalam data
  • Memberikan rekomendasi penyesuaian

Proses Keputusan:

  1. Deteksi nilai ekstrim (outlier)
  2. Evaluasi stabilitas parameter
  3. Bandingkan dengan range optimal
  4. Hitung penyesuaian yang diperlukan
  5. Berikan rekomendasi final

Tips Penting

  • Minggu 1: PPM rendah untuk perkembangan akar
  • Minggu 2-3: Tingkatkan PPM untuk pertumbuhan daun
  • Minggu 4: Kurangi PPM jika daun menguning
  • Jaga suhu air stabil

Koneksi ESP dengan Blynk:

  1. Buat akun di Blynk.io
  2. Buat project baru dan dapatkan Auth Token
  3. Gunakan token di konfigurasi ESP
  4. Tambahkan widget di aplikasi Blynk:
    • Gauge untuk PPM, pH, dan suhu
    • SuperChart untuk visualisasi data
    • Notification untuk peringatan
